# SlapChat Add

## Instructions

  1. Check out the SlapChat solution in this folder.
  2. Add a plus button and link it to a new message view controller. Name the class `FISAddMessageViewController`.
   - make its storyboardID `"addMessageVC"` 
  3. Add a save button and a text field. When you click save, save a new message with the contents of the text field to the context.
   - make your save button and text field accessibilty labels `"saveButton"` and `"messageField"`, respectively.
  4. When the table view controller is displayed, it should display the new message!
  5. **Don't forget to run the tests.**
